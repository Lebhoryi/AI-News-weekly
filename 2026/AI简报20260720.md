## 🔥 今日精选

🔴 **[月之暗面 / The Decoder]** Kimi K3 上线 48 小时算力见顶，暂停 C 端新用户订阅 — 7 月 19 日深夜，Kimi 团队发文称请求量逼近集群极限，即日起暂停新会员开通，存量算力优先保障已订阅用户，并推进扩容；后续或拆分主权益与 Kimi Code。2.8T 开源旗舰的「爆火」第一次以停售形式兑现，说明模型能力之外，分发与 GPU 供给才是真正瓶颈。（[中国网财经](http://finance.china.com.cn/industry/20260720/6316906.shtml)｜[The Decoder](https://the-decoder.com/moonshot-pauses-new-kimi-k3-subscriptions-after-gpu-demand-maxes-out-in-48-hours/)）

🔴 **[阿里 / The Decoder]** 阿里发布开源权重模型 Qwen 3.8（2.4T），自称「仅次于 Fable 5」 — 7 月 19 日 Qwen 团队推出 2.4 万亿参数多模态模型，称在编码与复杂生产力任务上超过 Qwen 3.7-Max；Token Plan / Qoder 预览价为标准价 10%，完整开源权重「即将」放出。时机紧贴 Kimi K3 热度，开源大战从「谁更大」转向「谁先把权重交出去、谁能稳住服务」。（[The Decoder](https://the-decoder.com/alibabas-qwen-takes-on-kimi-k3-with-open-weight-qwen-3-8-says-model-is-second-only-to-fable-5/)）

🔴 **[Anthropic]** Claude Fable 5 今日起按档位永久入订阅：Max / Team Premium 保留，Pro 改走额度 — 经多次延期后，7 月 20 日起 Fable 5 永久纳入 Max 与 Team Premium（上限为周用量 50%）；Pro / Team Standard 不再捆绑，获一次性 $100 用量额度，用尽后按 API（约 $10/$50 per MTok）计费。促销不确定期结束，前沿模型正式变成「订阅分层商品」。（[TechTimes](https://www.techtimes.com/articles/320905/20260718/claude-fable-5-ends-subscription-limbo-permanent-max-credits-only-pro.htm)）

## 🌍 国际动态

🟡 **[Google DeepMind]** GenCeption：用预训练视频生成模型做通用视觉感知 — DeepMind 基于阿里 Wan2.1，在单次前向中完成深度、分割、法向与 3D 姿态等任务，训练数据仅约 7500 段合成视频，多项基准接近专用模型。论文主张「视频生成器已内含世界模型表征」，与 LeCun 等对像素生成路线的批评形成对照。（[The Decoder](https://the-decoder.com/google-deepmind-argues-video-generators-already-contain-the-world-models-computer-vision-has-been-missing/)）

🟡 **[OpenAI / TechTimes]** GPT-5.6 Sol Full Access 误删文件与数据库事件持续发酵 — 7 月 19 日报道梳理：Sol 在 Full Access 模式下因 `$HOME` 展开等问题误删用户数据；OpenAI 曾承诺复盘，截至发稿官方 post-mortem 仍未公开，7 月 18 日 Codex 更新了捆绑指令或为部分静默修复。Agent 写权限扩大后，「会干活」与「会毁数据」成为同一枚硬币。（[TechTimes](https://www.techtimes.com/articles/320961/20260719/gpt-56-sol-deleted-files-databases-openai-had-63x-warning-it-ignored.htm)）

🟡 **[HN / Simon Willison]** Claude Code 运行时迁至用 Rust 写的 Bun — HN 热帖（约 500+ 赞）指出 Claude Code 已切换到 Rust 实现的 Bun 运行时，引发对 Agent CLI 性能、打包体积与生态兼容性的讨论。编码 Agent 的「外壳」本身也在快速工程化。（[Simon Willison](https://simonwillison.net/2026/Jul/19/claude-code-in-bun-in-rust/)｜[HN](https://news.ycombinator.com/item?id=48966569)）

🟡 **[Fortune / HN]** 美国电力公司动用土地征收权为数据中心拿地 — 报道称部分公用事业以「公共用途」征收土地服务 AI 数据中心，社区与法律争议升温。算力扩张正从芯片短缺外溢到土地与电网政治。（[Fortune](https://fortune.com/2026/07/19/data-center-eminent-domain-public-use/)）
