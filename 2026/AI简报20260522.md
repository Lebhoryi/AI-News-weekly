# 🤖 AI 早报 | 2026-05-22（周五）

> 📌 **今日一句话**：今天的 AI 动态继续围绕 Agent 展开：Google 把 ADK 推向 Kotlin 和 Android，Pydantic AI 发布 V2 Beta，Anthropic 则通过人才流动强化预训练研究。

---

## 🔥 今日精选

🔴 **[Google Developers Blog]** [ADK for Kotlin and ADK for Android 0.1.0](https://developers.googleblog.com/adk-kotlin-android-building-ai-agents/) — Google 发布 ADK for Kotlin 和 ADK for Android 0.1.0，让开发者在后端与 Android 应用中构建 agentic workflows。重点是混合编排：云端模型负责主控，设备端 Gemini Nano 子 agent 处理本地检索和隐私敏感任务。

🔴 **[GitHub]** [Pydantic AI v2.0.0b1](https://github.com/pydantic/pydantic-ai/releases/tag/v2.0.0b1) — Pydantic AI 发布 V2 Beta 1，引入 harness-first 设计，把 tools、生命周期 hooks、instructions、model settings 封装成可组合的 capabilities。Agent 框架正在从「一个 Agent 类」变成「能力模块拼装系统」。

🔴 **[Observer]** [Andrej Karpathy joins Anthropic](https://observer.com/2026/05/andrej-karpathy-joins-anthropic/) — Observer 报道，OpenAI 早期成员 Andrej Karpathy 加入 Anthropic 预训练团队，将参与用 Claude 加速下一代模型预训练研究。人才流向说明前沿模型竞争仍然押注预训练、数据和研究自动化。

## 🌍 国际动态

🟡 **[Google Developers Blog]** [ADK for Android](https://developers.googleblog.com/adk-kotlin-android-building-ai-agents/) — ADK for Android 支持 on-device sequential agents、本地检索、session state、memory service 和 OpenTelemetry。Google 把 Android 端 AI 从「调用模型」推进到「本地 agent 协作」。

🟡 **[Google Developers Blog]** [Antigravity CLI transition](https://developers.googleblog.com/en/an-important-update-transitioning-gemini-cli-to-antigravity-cli/) — Google 继续推进 Gemini CLI 向 Antigravity CLI 迁移，目标是统一命令行、IDE 和 agent-first 开发平台。CLI 正成为模型厂商争夺开发者入口的关键战场。

🟡 **[GitHub]** [GenericAgent](https://github.com/lsdefine/genericagent) — `lsdefine/genericagent` 是一个自进化 agent 项目，仓库介绍称核心种子约 3.3K 行，强调用更少上下文消耗扩展 skill tree。轻量 agent loop 仍是开源社区的活跃方向。

🟡 **[GitHub]** [Pydantic AI instrumentation changes](https://github.com/pydantic/pydantic-ai/releases/tag/v2.0.0b1) — Pydantic AI V2 Beta 将默认 instrumentation 升到 v5，并增加聚合 token usage span attributes。Agent 框架开始把可观测性当成默认能力，而非外部补丁。

🟡 **[Observer]** [Anthropic pretraining team expands](https://observer.com/2026/05/andrej-karpathy-joins-anthropic/) — 报道称 Karpathy 将向 Anthropic head of pretraining Nick Joseph 汇报，并组建团队研究用 Claude 加速预训练研究。模型公司正在尝试让当前模型参与下一代模型研发。

## 💡 今日洞察

Agent 基础设施正在分成两条线：一条是 Google ADK 这类平台型 SDK，把云端模型、设备端模型、工具和状态管理统一起来；另一条是 Pydantic AI、GenericAgent 这类开源框架，继续压缩核心 loop、强调可组合能力和可观测性。

更值得注意的是「模型参与模型研发」这条线。Karpathy 加入 Anthropic 预训练团队的意义，不只是人才流动，而是前沿实验室开始系统化探索用现有模型提高下一代模型研发效率。
