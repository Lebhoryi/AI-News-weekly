## 🔥 今日精选

🔴 **[量子位]** [τ0-WM：最大规模预训练的开源具身世界模型来了](https://www.qbitai.com/2026/05/426832.html) — 上海创智学院、智元机器人罗剑岚团队发布 τ0-World Model，使用约 3 万小时预训练数据，其中 1.78 万小时来自真机遥操作。它把“测试时计算”引入机器人动作选择，让机器人先模拟、评估、修正，再执行。

🔴 **[量子位]** [复旦×通义提出 ToolCUA](https://www.qbitai.com/2026/05/427005.html) — ToolCUA 面向 GUI+Tool 混合动作空间训练 Computer Use Agent，8B 模型在 OSWorld-MCP 上达到 46.85% 准确率，超过 Claude-4-Sonnet，接近 Claude-4.5-Sonnet。重点不是给 Agent 加更多工具，而是训练它判断什么时候该点界面、什么时候该调用工具。

🔴 **[The Decoder]** [AI search agents often confirm what they already know](https://the-decoder.com/ai-search-agents-often-confirm-what-they-already-know-instead-of-actually-researching-the-web/) — 哈工大与小红书研究者提出 LiveBrowseComp，显示静态浏览评测里很多高分来自模型记忆而非真实搜索；当问题依赖近 90 天低曝光事实时，模型排名会明显洗牌。这对 Agent 评测很关键：搜索能力需要动态、时效性 benchmark 才能测准。

## 🌍 国际动态

🟡 **[The Decoder]** [SoftBank 计划在法国建设 750 亿欧元级 AI 数据中心](https://the-decoder.com/softbank-plans-75-billion-euro-ai-data-center-buildout-in-france/) — 规划总容量最高 5GW，首期到 2031 年在法国北部建设 3.1GW、约 450 亿欧元设施。欧洲主权 AI 的瓶颈正在从模型能力转向电力、许可、机房和供应链。

🟡 **[The Decoder]** [Kaikaku.AI 发布多语言食材嵌入模型 Epicure](https://the-decoder.com/ask-ai-what-goes-with-chicken-and-the-answer-depends-on-whether-it-learned-from-recipes-or-molecules/) — Epicure 基于 414 万份、7 种语言食谱和 FlavorDB 化学数据训练，可区分“食谱里搭配什么”和“风味分子相似什么”。这是一个很垂直但有启发的例子：同一个查询，训练数据决定模型回答的文化与物理语义。

🟡 **[The Decoder]** [Anthropic 在招聘面试中禁用 AI 工具](https://the-decoder.com/anthropic-bans-ai-tools-during-job-interviews-to-see-how-candidates-actually-think/) — 报道称 Anthropic 为了观察候选人的真实思考过程，在面试环节限制 AI 使用。AI 公司自己也在重新划定“人类能力评估”和“AI 辅助工作”的边界。

🟡 **[The Decoder]** [Anthropic 研究显示社科研究者使用 AI coding agent 存在明显性别差异](https://the-decoder.com/anthropic-study-finds-men-use-ai-coding-agents-more-than-twice-as-often-as-women-in-social-science-research/) — 报道称男性姓名研究者使用 coding agent 的频率超过女性姓名研究者两倍，且在同学科、同职业阶段仍存在差距。这提示 AI 工具扩散可能放大既有技能与机会分布不均。

⚪ **[量子位]** [亚马逊王晓野：Token 贵，往往因为喂给模型的垃圾太多](https://www.qbitai.com/2026/05/426970.html) — 这类观点正在成为企业 Agent 落地的实用共识：成本优化不只是换便宜模型，也包括上下文治理、任务拆分和检索质量控制。

## 💡 今日洞察

今天的主线很清楚：Agent 的瓶颈不再只是“有没有工具”，而是能否在长程任务里做正确路径选择，并用证据修正自己的判断。机器人世界模型也在走同一条路，从端到端反射式动作，转向测试时模拟、评估和纠错。

另一个信号是评测正在被重新审视。静态 benchmark 越来越容易混入参数记忆，真正有区分度的评测会更依赖时间窗口、真实环境和可验证轨迹。
