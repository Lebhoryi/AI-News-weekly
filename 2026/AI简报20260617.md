## 🔥 今日精选

🔴 **[Reuters / Bloomberg / The Decoder]** [SpaceX 600 亿美元收购 Cursor 开发商 Anysphere](https://the-decoder.com/spacex-bets-60-billion-on-cursor-to-catch-openai-and-anthropic/) — 6 月 16 日，SpaceX 正式敲定对 AI 编程助手 Cursor 母公司 Anysphere 的收购，投资者以 SpaceX 股票换股，预计 2026 年 Q3 交割。Cursor 年化收入约 30 亿美元、超 3000 家客户年付 10 万美元以上；xAI 借此补齐 coding agent 短板，Cursor 则获得 SpaceX 芯片储备。**背景补充**：Anysphere 即 Cursor 背后的公司，此前主要调用 OpenAI / Anthropic 模型，收购后 xAI 将与 Cursor 联合训练新模型。

🔴 **[The Information / Reuters / The Decoder]** [DeepSeek 首次引入外部融资，估值超 500 亿美元](https://the-decoder.com/deepseek-takes-outside-money-for-the-first-time-at-a-50-billion-valuation/) — 据 The Information，DeepSeek 完成首轮融资，募资超 500 亿元人民币（约 74 亿美元），估值突破 500 亿美元；4 月传闻仅为 10 亿美元估值、3 亿美元融资。结构特殊：多数资金须通过 CEO 梁文锋管理的有限合伙注入，无投票权且锁定期 5 年；腾讯、宁德时代等参投，国家 AI 产业投资基金直投并保留投票权。V4 Pro 永久 75 折，输入价约为 GPT-5.5 的 1/11。

🔴 **[Ed Zitron / Financial Times / The Decoder]** [OpenAI 去年烧掉 340 亿美元，净亏损约 390 亿](https://the-decoder.com/openai-burned-through-34-billion-last-year/) — FT 独立确认：研发约 190 亿、销售营销近 60 亿，年收入约 130 亿；年底月收入 20 亿（2024 年末为每季度 10 亿）。剔除约 300 亿一次性非现金会计调整后，经营性亏损约 80 亿。公司正筹备估值或超 1 万亿美元的 IPO。

## 🌍 国际动态

🟡 **[The Decoder / WSJ]** [Anthropic 撤回 Claude Agent SDK 计费改革](https://the-decoder.com/anthropic-backs-off-unpopular-billing-overhaul-as-price-war-with-openai-looms/) — 原定于 6 月 15 日将 Agent SDK、`claude -p` 与第三方应用移出订阅额度、改用量计费；Anthropic 宣布「暂时不变」。背景：OpenAI 据报酝酿 API 大幅降价、Anthropic 筹备 IPO，且 Fable 5 出口管制余波未平，激进改价恐流失开发者。

🟡 **[Axios / The Decoder]** [Microsoft Copilot Cowork 转用量计费，或接入自托管 DeepSeek V4](https://the-decoder.com/microsofts-copilot-cowork-moves-to-usage-based-billing-and-may-tap-deepseek/) — Copilot 执行副总裁 Charles Lamanna 称 flat-rate 难覆盖高频 agent 用户；Cowork 基于 Claude，token 消耗大。微软考虑在 Azure 上部署微调版 DeepSeek V4 作为低成本选项，数周内定案。

🟡 **[Claude Code Changelog]** [Claude Code 2.1.179 发布](https://code.claude.com/docs/en/changelog#2-1-179) — 6 月 16 日更新：修复 bash 工具 `run_in_background` 参数、改进 Remote Control 会话管理、优化 MCP OAuth 认证流程等。

🟡 **[Berlin Court / The Decoder]** [柏林法院：Google AI Overviews 属搜索新格式，非原创内容](https://the-decoder.com/berlin-court-rules-googles-ai-overviews-are-just-a-new-search-format-not-original-content/) — 德国出版商诉 Google AI 摘要侵权案获阶段性裁决；法院认定 AI Overviews 是搜索结果呈现方式，不单独构成版权作品，对全球 AI 搜索版权诉讼有参照意义。

🟡 **[HN]** [Running local models is good now](https://news.ycombinator.com/item?id=48555993) — 1040 分热帖：Fable 5 全球下线后，开发者重新评估本地 / 开源模型在 coding workflow 中的可用边界，与 SpaceX 重金押注 Cursor 形成有趣对照。

## 💡 今日洞察

SpaceX 600 亿买 Cursor 与 DeepSeek 500 亿估值同日出现，说明 AI 价值正分裂为两条主线：西方闭源巨头通过并购抢 coding agent 入口（xAI 借 Cursor 补 talent + compute），中文开源模型则靠价格与开放权重重估资本故事—— DeepSeek V4 Pro 输入价仅为 GPT-5.5 的约 1/11，已直接牵动 Microsoft 是否在 Copilot Cowork 中引入 DeepSeek。
OpenAI 340 亿 burn rate 与 Anthropic 撤回计费改革同框，API 价格战已从「降价抢市场」进入「保用户 vs 保 IPO 估值」阶段；企业侧 flat-rate 订阅与 agent 高 token 消耗之间的结构性矛盾，将迫使更多产品转向混合计费。
HN 热帖「本地模型够用了」与 Cursor 被天价收购并存，短期看云端 frontier 仍是生产力首选，但 Fable 下线与地缘断供风险正在加速开发者评估 hybrid 架构—— 这与 GitHub 上 Agent-Reach、cua 等 Agent 基础设施的持续高热形成呼应。
